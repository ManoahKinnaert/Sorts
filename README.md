# SORTS
If you haven't got things sorted out yet... this is your chance.
This repo contains some experiments regarding sorting algorithms. These experiments attempt to measure 
the amount of compares and exchanges with regard to the input size given to the sorting algorithm (Insertion sort or Quicksort).

# How to get going

1) Create a virtual environment and activate it (Unix / Linux).
```bash
python -m venv venv && source venv/bin/activate
```

2) Update / Upgrade pip to the latest version.
```bash
pip install --upgrade pip
```

3) Install requirements.
```bash
pip install -r requirements.txt
```

4) Run the desired experiment(s).
```bash
python code
```
